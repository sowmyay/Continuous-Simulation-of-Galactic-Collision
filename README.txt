Required Packages
--------------------------------------------------------------------------------------
In order to run the simulation, the following packages are required:

NumPy
SciPy
MatPlotLib


Execution
--------------------------------------------------------------------------------------
To run the simulation, run SimulationModel.py in the following manner:

python SimulationModel.py

Input.txt contains :
1. Mass of the main galaxy (in Solar masses) = M
2. Mass of the disruptor galaxy (in Solar masses) = S
3. Total time of the simulation (in parsecs) = T
4. Number of time steps = n

n number of position scatter plots will be generated, where the the red dots indicate the stars of the main galaxy and the blue dot represents the disruptor galaxy
